# 邻里币规则

## 基本概念
- 将现有的「信用分」改为「邻里币」
- 新用户初始 0 邻里币
- 邻里币只增不减（惩罚走违规体系）

## 获取规则

| 行为 | 奖励 | 每日上限 | 说明 |
|------|------|----------|------|
| 发布闲置/免费送 | +5 | 3次/天（15币） | 鼓励邻里交易 |
| 发帖（论坛/互助/拼车/团购/宠物） | +3 | 5次/天（15币） | 鼓励社区互动 |
| 邀请好友入驻 | +20 | 无上限 | 被邀请人完成社区认证后发放 |
| 每日首次登录 | +1 | 1次/天 | 提升日活 |
| 交易完成（标记已售） | +10 | 无上限 | 鼓励真实成交 |

## 等级体系

| 等级 | 所需邻里币 | 称号 |
|------|-----------|------|
| Lv.1 | 0 ~ 49 | 新住户 |
| Lv.2 | 50 ~ 199 | 活跃邻居 |
| Lv.3 | 200+ | 金牌邻居 |

## 防刷机制
- 发布后删除不扣币，但同一内容重复发布不重复奖励
- 每日获取有上限，防止刷币
- 邀请奖励在被邀请人完成注册后才发放

## 兑换规则（后续实现）

### 兑换档位

| 档位 | 所需邻里币 | 可兑换物品示例 |
|------|-----------|---------------|
| 小件 | 50 币 | 垃圾袋、纸巾、湿巾、洗碗海绵 |
| 中件 | 150 币 | 洗衣液、洗洁精、牙膏套装、收纳盒 |
| 大件 | 300 币 | 毛巾礼盒、保温杯、雨伞、充电线 |
| 精选 | 500 币 | 小风扇、蓝牙音箱、厨房秤、便携榨汁杯 |

### 兑换规则
- 每人每月限兑 3 次，防止囤积刷换
- 兑换后邻里币即时扣除，不可退还
- 物品由物业/社区提供，线下到物业处领取
- 库存为 0 时自动下架，补货后重新上架

### 运营思路
- 物品由物业采购，成本低但实用，提升用户对社区的归属感
- 后期可引入周边商家赞助礼品，商家获得曝光，平台零成本
- 节假日可推限时兑换活动（如中秋月饼礼盒 200币）
