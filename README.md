# 🏘 邻里市集 - 小区二手交易微信小程序

## 项目简介

邻里市集是一个面向单个小区内部用户的本地化二手交易与社区互动微信小程序。

## 功能模块

| 模块 | 说明 |
|------|------|
| 🏠 首页 | 小区选择、广告轮播、快捷入口、商品瀑布流、分类筛选 |
| 📦 发布 | 发布闲置商品、免费送、社区帖子、求帮忙 |
| 💬 消息 | 私信聊天、系统通知 |
| 👤 我的 | 个人信息、楼栋认证、信用体系、发布/收藏管理 |
| 🗣 楼里事儿 | 社区论坛（公告/吐槽/求助/活动） |
| 🤝 邻里帮忙 | 借物/拼车/求推荐，支持紧急标签 |
| 🏠 房屋租赁 | 房源发布，区分房东直租/中介 |
| 🔍 搜索 | 关键词搜索、搜索历史、热门搜索 |

## 快速开始

### 1. 安装微信开发者工具
下载地址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

### 2. 导入项目
1. 打开微信开发者工具
2. 选择「导入项目」
3. 选择本项目文件夹
4. AppID 填入你的小程序 AppID（或使用测试号）
5. 点击「导入」

### 3. 运行
导入后项目会自动编译运行，可在模拟器中预览所有页面。

## 技术说明

- **前端框架**：微信小程序原生开发
- **数据**：当前使用本地 Mock 数据，可替换为真实后端 API
- **样式**：自定义 UI 组件，微信绿色主题

## 后端对接

项目中所有数据来自 `utils/mockData.js`，对接真实后端时：

1. 在 `app.js` 的 `globalData.baseUrl` 中配置后端 API 地址
2. 将各页面中的 Mock 数据调用替换为 `wx.request` 请求
3. 推荐后端方案：
   - 微信云开发（最快上手）
   - Node.js + MySQL
   - Supabase

## 注意事项

- `project.config.json` 中的 `appid` 需替换为你的真实小程序 AppID
- 图片资源为占位图，正式使用时请替换为实际设计图
- 登录功能需要配置微信开放平台相关权限
